# GitHub かんたん使い方ガイド（超初心者向け）

このページは「GitHubを触ったことがない人」向けの説明です。
専門用語はできるだけ使わず、ファイルのダウンロードと差し替えだけ分かれば使えるようにまとめています。

---

## 1. GitHubってなに？

GitHub（ギットハブ）は、ファイルをインターネット上に置いて、みんなで共有できる無料のサービスです。
このリポジトリ（=ファイルの保管場所）では、プロジェクト管理用のExcelテンプレートを配布しています。

- **アカウントは不要**：ダウンロードするだけなら、ログインも会員登録もいりません。
- **無料**：費用はかかりません。

---

## 2. ファイルをダウンロードする方法（いちばん大事）

最新のExcelファイルは `Latest` フォルダに入っています。

1. リポジトリのトップページを開く
2. `Latest` フォルダをクリックする
3. `PM_Template.xlsx` をクリックする
4. 右上のほうにある **Download（または Raw）** ボタンを押す
5. パソコンに `PM_Template.xlsx` が保存される

> 💡 もし中身が文字化けのように見えるページが開いても、あわてず **Download** ボタンを押せばファイルが保存されます。

---

## 3. リンクをそのまま配布したいとき

note などに「ここから直接ダウンロード」と書きたい場合は、次のリンク（raw形式）を貼ると、
クリックするだけでファイルがダウンロードされます。

```
https://raw.githubusercontent.com/KazunoriIwata/project-management-templates/main/Latest/PM_Template.xlsx
```

> ※このリンクは、Latestフォルダに `PM_Template.xlsx` がアップロードされた後に使えるようになります。

---

## 4. フォルダの意味

| フォルダ | 中身 |
|---|---|
| Latest | 最新版のExcelファイル |
| Archive | 過去のバージョン |
| Manual | この使い方ガイドなどの説明 |
| Images | 説明用の画像・スクリーンショット |

---

## 5. ファイルを新しくする方法（管理者向け）

テンプレートを更新したいときの手順です。GitHubにログインした状態で行います。

1. `Latest` フォルダを開く
2. 右上の **Add file ▸ Upload files** をクリック
3. 新しい `PM_Template.xlsx` をドラッグ＆ドロップ（**同じファイル名にすると上書き**されます）
4. 下にある緑色の **Commit changes** ボタンを押す

> 同じファイル名で上書きすれば、配布しているリンクはそのまま使えます（リンクの貼り替え不要）。
> 過去版を残したいときだけ、`Archive` フォルダに `PM_Template_Ver1.0.xlsx` のように別名で保存してください。

---

## 6. アップロード前のチェック（管理者向け・重要）

公開リポジトリなので、Excelをアップロードする前に必ず確認してください。

- 個人情報・顧客情報・社外秘が入っていないか
- 隠しシートがないか
- ファイルの作成者情報（会社名・個人名）が残っていないか
- パスワードや外部データ接続が残っていないか

---

困ったときは、このガイドの手順 **2** だけ覚えておけば、ファイルのダウンロードはできます。
