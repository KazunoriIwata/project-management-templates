# Project Management Templates

プロジェクト管理・WBS・レビュー管理などに使えるExcelテンプレート集です。

## ダウンロード

最新版は `Latest` フォルダからダウンロードしてください。

- Latest/PM_Template.xlsx

## フォルダ構成

- Latest：最新版ファイル
- Archive：過去バージョン
- Manual：利用ガイド
- Images：説明画像・スクリーンショット

## 更新履歴

- Ver1.0：初回公開

## 利用方法

1. `Latest` フォルダを開く
2. Excelファイルを選択
3. 右上の Download または Raw からダウンロード
